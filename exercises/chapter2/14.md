# 练习题 2-14

## 练习
> 假设 x 和 y 的 字节值分别为 0x66 和 0x39。填写下表，指明各个 C 表达式的字节值。

| 表达式   | 值        |
| -------- | --------- |
| x & y    | 0010 0000 |
| x \| y   | 0111 1111 |
| ~x \| ~y | 1101 1111 |
| x & !y   | 0000 0000 |
| x && y   | 0000 0001 |
| x || y   | 0000 0001 |
| !x || !y | 0000 0000 |
| x && ~y  | 0000 0001 |
